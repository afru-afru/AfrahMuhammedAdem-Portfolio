Description:

I developed a Q&A feature for the Evangadi Company website, which serves as a learning platform for users. This forum allows users to ask questions, share knowledge, and engage in discussions, fostering an interactive community.

Technologies Used:

Front-End:Reactjs
Backend:Nodjs and Express
Database: MySQL

Key Features:

User Authentication: Secure sign-up and login processes for users to manage their profiles.
Question and Answer Posting: Users can easily post questions and provide answers, with the ability to comment and vote on responses.
Edit and Delete Functionality: Users can edit or delete their questions and answers, allowing for easy updates and corrections.
View Functionality: Users can view their own questions and answers, as well as those from other users, enhancing engagement.
